# Changelog

## v1.0.0 - 2026-07-10

- Initial DrawDash Android release.
- Added timed drawing matches with 30, 60, 90, and 120 second modes.
- Added responsive Compose home, game, settings, statistics, tutorial, pause, and results screens.
- Added stroke-based drawing canvas with pencil, eraser, undo, redo, clear, colour, stroke size, and skip.
- Added local deterministic drawing recogniser behind a modular interface.
- Added simulated opponents with difficulty-based scoring profiles.
- Added DataStore persistence for settings and statistics.
- Added bundled word database and automated tests.
- Added CI and signed GitHub Release workflow.
